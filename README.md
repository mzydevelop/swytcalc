# swytcalc
喜闻乐见的事情来了，三位一体综合评价计算器开源了！本计算器开源协议为GPL（GNU GENERAL PUBLIC LICENSE）。
最新版本的Build为0014（20170326更新）！
开发源码为Visual Basic 6

版本改进：


0223版本更新内容

1.提供了第二窗体计算页面一键切换学校功能

2.支持在软件第一界面打开招生章程（切换时请先输入其他学校序号按查询按钮后再按招生章程按钮）

3.增加招生章程，报名系统网址一键打开功能，新增报名时间查看功能。

4.修复多处bug，提高稳定性。

5.增加了对照表

0318（Build0010版）版本更新内容

1.增加官网访问项，

2.添加了所有学校的报名系统网址，

3.新增下载最新版计算器功能。
本计算器只计算普通类，上列三位一体院校有艺术体育类的均没有计算方式采取的都是普通类的算法。
目前已支持第二页面一键切换学校，感谢榕姐姐的使用反馈。

0326（Build0014版）版本更新内容

最新增加了学考信息导出到config.ini,并对关于窗体加以完善，进行小幅修改。

本计算器最新版为20170326下午正式版，build号为0014
